# Vim in zsh shell

## How to enable

You will need to add the following to .zshrc.

```zsh
bindkey -v
```

1. Open a terminal
2. Type `vim ~/.zshrc` or use any other editor to open the `.zshrc` file
3. Add the following

```zsh
bindkey -v
```

4. Restart zsh shell

## Available Vim key bindings

`i` `a` `I` `A`

`$` `0` `w` `b` `W` `B` `e` `E`

`d` `c`
